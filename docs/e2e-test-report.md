## 测试场景

### 场景 1: 完整旅行咨询对话

**测试流程**:
```
1. 用户点击连接 → WebSocket 连接建立
2. 用户说话（语音输入）→ 音频流发送
3. STT 识别 → 语音转文本
4. 发送 travel-agency → Agent 处理
5. 接收 Agent 回复 → 文本响应
6. TTS 合成 → 文本转语音
7. 播放音频 → 用户听到回复
8. 显示字幕（用户+AI）→ 可视化反馈
9. 挂断连接 → 断开 WebSocket
```

**预期结果**:
- ✅ 所有步骤成功执行
- ✅ 端到端延迟 < 7s
- ✅ 字幕正确显示

**测试结果**: ✅ **通过**
