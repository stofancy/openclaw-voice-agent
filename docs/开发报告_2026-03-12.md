### 3. 性能优化 ✅

**改动文件**: `wsl2/agent-gateway.py`

**优化措施**:

#### 3.1 TTS 预初始化
```python
# 客户端连接时预初始化 TTS
if not self.tts_pre_initialized:
    self.init_tts()
    self.tts_pre_initialized = True
```

**效果**: 避免首次 TTS 调用时的连接延迟 (~1-2s)

#### 3.2 并行处理
- STT 识别过程中保持 TTS 连接就绪
- Agent 处理时检查 TTS 状态，确保快速响应

#### 3.3 流式 STT
- 实时发送音频到百炼，无需等待说话结束
- 增量识别结果实时更新字幕

**预期延迟优化**:
```
优化前：6-10s
优化后：<5s (目标)

分解:
- STT: ~1-2s (流式，首字<2s)
- LLM: ~2-3s (Agent 响应)
- TTS: ~1-2s (预初始化后)
```
